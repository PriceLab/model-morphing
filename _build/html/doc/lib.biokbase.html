lib.biokbase.log module¶

NAME
log
DESCRIPTION
A library for sending logging messages to syslog.
lib.biokbase.log.log(string subsystem, hashref constraints)[source]¶

Initializes log. You should call this at the beginning of your program. Constraints are optional.

lib.biokbase.log.log_message(int level, string message)¶

sends log message to syslog.

* level

(0-9) The logging level for this message is compared to the logging level that has been set in log. If it is <= the set logging level, the message will be sent to syslog, otherwise it will be ignored. Logging level is set to 6 if control API cannot be reached and the user does not set the log level. Log level can also be entered as string (e.g. ‘DEBUG’)

* message

This is the log message.

lib.biokbase.log.get_log_level()¶

Returns the current log level as an integer.

lib.biokbase.log.set_log_level(integer level)¶

Sets the log level. Only use this if you wish to override the log levels that are defined by the control API. Can also be entered as string (e.g. ‘DEBUG’)

* level

priority

* 0

EMERG - system is unusable

* 1

ALERT - component must be fixed immediately

* 2

CRIT - secondary component must be fixed immediately

* 3

ERR - non-urgent failure

* 4

WARNING - warning that an error will occur if no action is taken

* 5

NOTICE - unusual but safe conditions

* 6

INFO - normal operational messages

* 7

DEBUG - lowest level of debug

* 8

DEBUG2 - second level of debug

* 9

DEBUG3 - highest level of debug

lib.biokbase.log.set_log_msg_check_count(integer count)¶

used to set the number the messages that log will log before querying the control API for the log level (default is 100 messages).

lib.biokbase.log.set_log_msg_check_interval(integer seconds)¶

used to set the interval, in seconds, that will be allowed to pass before log will query the control API for the log level (default is 300 seconds).

lib.biokbase.log.update_api_log_level()¶

Checks the control API for the currently set log level.

lib.biokbase.log.use_api_log_level()¶

Removes the user-defined log level and tells log to use the control API-defined log level.
